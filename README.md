# tozan-ecc

![Go](https://github.com/armfazh/tozan-ecc/workflows/Go/badge.svg)

Elliptic curve operations in Go.

### Features

Curve models supported:
 -   Weierstrass
 -   Montgomery
 -   Twisted Edwards
